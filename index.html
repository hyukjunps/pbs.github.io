1
가사 옆에 조명 번호를 괄호로 적어주세요
노래 또는 가사 옆에 사용할 조명 번호를 적어주세요. 색 이름이 아닌 번호로 적어야 해요.
✓어떻게 이별까지 사랑하겠어. 널 사랑하는거지 (1+10)
✗어떻게 이별까지 사랑하겠어. 널 사랑하는거지 (빨간색)
이 가사에서 파란색(10번) 조명을 사용한다는 뜻이에요
2
박자 빠른 곡에서 계속 색 바꾸기는 어려워요
박자가 빠른 곡에서 매 소절마다 완전히 다른 색으로 바꾸는 건 어렵지만, 두 색을 번갈아 쓰거나 사이키를 활용하면 가능해요.
✗Cheer up baby (1+10) Cheer up baby (1+6) 좀 더 힘을 내 (1+9)
세 가지 완전히 다른 색으로 계속 바꾸는 건 어려워요
✓Cheer up baby (14) Cheer up baby (11+14) 좀 더 힘을 내 (1+11)
사이키(14)와 핑크 번갈아 — 조명 상황에 따라 가능해요
✓Cheer up baby (1+6) Cheer up baby (1+7) 좀 더 힘을 내 (1+6)
빨강(6)↔초록(7) 두 색 번갈아 사용 — 가능해요
✓Cheer up baby (1+6) Cheer up baby (1+6+7) 좀 더 힘을 내 (1+6)
빨강과 빨강+초록 번갈아 — 가능해요
